# IP Studio - AI 驱动的自创 IP 形象创作与运营 SaaS

为想打造并自主运营 IP 形象的创作者，提供从基础草图到完整形象、运营内容与可售周边的一站式 AI 工作台。

## 本质

IP Studio 不是单纯的 AI 绘图工具，也不是普通社媒内容生成器；它是面向「自创 IP → 持续运营 → 变现」全链路的 IP 经营系统。产品把「形象长什么样、怎么变体、发什么内容、卖什么周边」收敛成一条可由 AI 协同完成的创作与运营流程。

MVP 聚焦形象侧：用户上传或手绘基础图，AI 产出 IP 三视图与高质量形象，并基于三视图扩展换装形象、运营图文与视频，支持下载使用。周边定制与售卖为后续阶段。

## 商业模式

订阅制，按创作与运营强度分层：

- 49 元档：轻量 IP 创作者，满足基础形象生成与少量内容产出。
- 99 元档：稳定运营者，满足三视图扩展、换装变体与更高频的图文、视频产出。
- 199 元档：重度 IP 运营者，满足规模化形象变体、多形态运营内容与周边制作需求。

## AI 角色

AI 是用户的 IP 创作搭档与运营助手。它帮助用户：

- 从上传或手绘基础图，生成完整、好看的 IP 三视图与形象资产。
- 基于核心形象扩展换装变体、运营图文、视频与文案等社交媒体 IP 运营产物。
- 支撑用户在小红书等平台持续运营自创 IP，并为后续周边制作提供形象基础。

## 技术栈

全栈 TypeScript，顶层由 app/chat/api 三个 L1 模块组成：

- app: Bun + Vite + React + React Router + Tailwind CSS v4。
- chat: Bun + Hono + TypeScript + Mastra + zod。
- api: Bun + Hono + TypeScript + zod + Supabase client。
