# app/
> L1 | 父级: /CLAUDE.md

模块定位: 用户端，使用 Bun + Vite + React + React Router + Tailwind CSS v4 承载 SaaS 浏览器体验。

成员清单
src/app/: 应用装配边界，放置前端运行时外壳、路由挂载与全局 Provider 职责说明。
src/pages/: 页面级路由边界，按 URL 组织用户可访问页面职责说明。
src/widgets/: 业务组件组合层，承载跨页面可复用的 UI 片段职责说明。
src/shared/: 前端共享层，承载通用 UI、工具、常量与类型职责说明。
